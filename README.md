# TipiTip brand assets

Logo marks, lockups, and animated logos for [TipiTip](https://tipitip-sable.vercel.app) (per-paragraph cUSD tipping on Celo).

## Files

| File | What |
|------|------|
| `tipitip-icon-dark.png` | square app icon, navy bg (1024px) |
| `tipitip-icon-cream.png` | square app icon, cream bg (1024px) |
| `tipitip-icon-rose.png` | square app icon, rose bg (1024px) |
| `tipitip-lockup-cream.png` | mark + wordmark, cream (1600px) |
| `tipitip-lockup-dark.png` | mark + wordmark, dark (1600px) |
| `tipitip-logo-cycle.gif` | colour-cycling logo (navy / rose / plum / teal / blue / violet) |
| `tipitip-logo-wbr.gif` | white / black / red logo |

The mark is a piece of movable type (a paragraph mark cast in a type sort); the wordmark is set in IBM Plex Mono Light with the three i-dots as coins.
